# Quiz 2 Study Guide

## Learning Objectives

This quiz will cover the concepts we've learned so far, including Flask routes, route variables, forms, templates, APIs, and JSON. You may want to review the [Quiz 1 Study Guide](quiz-1-study-guide.md) as the information presented there will be covered on this quiz.

In addition, you should be able to...

- Use the `requests` library to make an API request and combine it with a Flask route, as shown in [Module 3, Lesson 6: Using the Requests Library to Generate User Requests](../Lessons/module-3/lesson-6.md)
- Pass parameters to an API call as key/value pairs
- Use dictionary syntax to access the returned JSON data from an API, as shown in [Module 3, Lesson 3: Working with JSON Objects](../Lessons/module-3/lesson-3.md) and as practiced in [Homework 4: APIs](../Assignments/03-APIs.md)
